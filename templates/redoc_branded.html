Respectlytics REST API Reference

Explore endpoints, request and response schemas, authentication details, and integration guidance for the Respectlytics API. The interactive documentation loads below when JavaScript is available.

Quick Links

  • OpenAPI Schema (JSON): /api/v1/reference/schema.json
  • OpenAPI Schema (YAML): /api/v1/reference/schema.yaml
  • Interactive API Explorer: /api/v1/reference/swagger/

Authentication

API requests use your App Key in the X-App-Key header. Create and manage keys from your dashboard settings.

Primary Endpoints

  • POST /api/v1/events/ — Ingest analytics events.
  • POST /api/v1/events/batch/ — Ingest events in batch.
  • GET /api/v1/stats/ — Retrieve aggregated analytics metrics.
  • POST /api/v1/events/delete/ — Delete events in a selected date range with optional filters.
